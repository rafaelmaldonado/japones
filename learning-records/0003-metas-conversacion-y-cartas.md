# 0003 — Metas concretas: conversación y cartas; repo real

**Fecha:** 2026-07-16

## Contexto
El estudiante precisó la misión: quiere (1) **sostener una conversación** y
(2) **escribir cartas** en japonés. También indicó que 3 lecciones es poco:
espera un temario amplio. Proporcionó el repo real:
https://github.com/rafaelmaldonado/japones.git

## Decisiones
- Misión actualizada con las dos metas concretas.
- Se define un **temario largo por unidades** (7 módulos, decenas de lecciones):
  escritura → supervivencia → gramática → vocabulario → conversación → kanji →
  cartas.
- `course.js` pasa a organizarse por **unidades** con muchas lecciones
  (la mayoría "locked" hasta crearse), para que el índice muestre el camino
  completo y motive.
- README ajustado al usuario real `rafaelmaldonado`, repo `japones`.

## Zona de desarrollo próximo
Seguir con Unidad 1 (escritura): Lección 2 = columna K. El grueso del temario
está bloqueado y se irá desbloqueando lección a lección.
