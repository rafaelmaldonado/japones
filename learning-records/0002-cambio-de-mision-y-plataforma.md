# 0002 — Cambio de misión, español y GitHub Pages

**Fecha:** 2026-07-16

## Contexto
En la primera sesión la misión era "progreso constante" sin meta concreta.
El estudiante aclaró que sentirá progreso cuando pueda **escribir, entender y
hablar** japonés — las tres destrezas, no solo lectura.

## Decisiones
- **Misión actualizada** (confirmado por el estudiante): éxito = escribir +
  entender + hablar. Se añade práctica de trazos de kana y de pronunciación,
  además del reconocimiento visual.
- **Idioma de enseñanza: español.** Todo el material se reescribe en español.
- **Plataforma: GitHub + GitHub Pages.** Requiere:
  - `index.html` de portada que lista las lecciones.
  - Navegación en cada lección para volver al índice.
  - Barra de progreso visible.
  - Rutas relativas (ya se usaban).
- Se reescribieron: `style.css`, Lección 1, chart de kana; y se creó `index.html`
  y el componente de navegación/progreso.

## Zona de desarrollo próximo
Siguiente lección: columna K (か き く け こ) con trazos + audio de pronunciación,
intercalando las vocales.
